১. বিনিয়োগ ঝুঁকি: InvestBD তে বিনিয়োগ করা অর্থের উপর ঝুঁকি থাকতে পারে। আপনার নিজের দায়িত্বে বিনিয়োগ করুন।

২. Account নিরাপত্তা: আপনার account এর পাসওয়ার্ড কাউকে শেয়ার করবেন না। নিজের account এর নিরাপত্তা আপনার দায়িত্ব।

৩. Deposit ও Withdraw: সঠিক Transaction ID না দিলে Deposit অনুমোদিত হবে না। Withdraw এর জন্য ২৪ ঘণ্টা পর্যন্ত সময় লাগতে পারে।

৪. Daily Tasks: প্রতিদিনের Task সম্পন্ন করলেই আয় যোগ হবে। Task না করলে কোনো আয় হবে না।

৫. Referral: Referral বোনাস শুধুমাত্র আপনার referral link বা code ব্যবহার করে registration ও plan কেনার পরে যোগ হবে।

৬. Account বাতিল: যেকোনো প্রতারণামূলক কার্যক্রমের জন্য account বাতিল করার অধিকার InvestBD সংরক্ষণ করে।

৭. পরিবর্তন: InvestBD যেকোনো সময় এই শর্তাবলী পরিবর্তন করার অধিকার রাখে।
